Курс читался в «Нижегородском институте информационных технологий»(НИИТ) в рамках комплексной программы "Разработка программного обеспечения". Содержание курса:
- Причины использовать Python, плюсы, недостатки и отличия языка, сценарии использования, введение в интерпретатор, выполнение программы, окружение (версии, pip, pep8, virtualenv).
- Основы, комментарии, динамическая типизация, типы, операции, условия, циклы, исключения.
- Функции (в том числе lambda и встроенные), области видимости, генераторы, декораторы, итераторы, списковые выражения.
- ООП, магические методы, протоколы (with, iter и прочие), self, cls, метапрограммирование.
- Импорты, пакеты, файловые операции, создание собственного пакета, создание консольного интерфейса.
- IDE (pycharm), debug, тестирование (unittest, pytest).
- Треды, процессы, ассинхронность, concurrency, профилирование.
- Регулярные выражения, сериализация (сvs, json, yaml, xml).
- Возможности стандартной библиотеки 1.
- Возможности стандартной библиотеки 2.
- WEB фреймворки.
- БД, ORM модели и SQL алхимия.
- Диспетчеры сообщений, распределенные системы управления очередями задач, встраивание их в систему.
- Системы делоя (salt и прочие).
- Расширение и взаимодействие с другими языками, обзор часто используемых внешних библиотек и пакетов.
- Пакеты для научных расчетов и работы с графикой.
